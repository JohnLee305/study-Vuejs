###[2022-04-18]
- MVVM 모델
  - MVC와 다르게 뷰모델로 되어있음
  - View Model 은 구조상으로 모델과 뷰 사이에 위치함.
  - 모델에서 데이터를 가지고 오는데 이 데이터는 뷰에서 사용하기 적합한 형태로 가지고 온다.
  - 뷰에서 조금더 적극적으로 사용할 수 있게 해줌,
  - 모델과 뷰모델은 1 : 1로 매칭되기 때문에 뷰가 많은경우에는 여러개의 뷰모델을 구성할 수 있다.
  - 뷰 모델은 데이터가 변경될때마다 즉시 화면에 반영이 된다.
